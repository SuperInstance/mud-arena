# CHARTER — MUD Arena

## Mission
GPU-accelerated MUD for agent script backtesting. Agents write scripts, GPUs run millions of scenarios, LLMs generate challenges, scripts evolve to be more clever with less agent intervention.

## Type
vessel

## Key Innovation
Backtesting for agent behavior. Like quant finance backtests trading strategies, this backtests agent scripts against simulated worlds at GPU speed.

## The Insight (from Casey)
"Scripts get more clever, agents get more passive. The game plays itself while the agent coaches from the sidelines."
